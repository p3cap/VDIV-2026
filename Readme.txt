======== Setup ========
- Telepísd a Python 3.13 vagy újabb verziót. (https://www.python.org/downloads/)
- Telepísd a node.js-t. (https://nodejs.org/en/download)
- Telepísd a c++ (opcionális, gyorsabb pathfinding-hoz ajánlott)

======== Futtatás ========

-- macOS / linux --
Telepísd le a függőségeket: `python3 setup_deps.py`
Program indítása: `python3 run_dev_linux.py`

-- Windows --
Telepísd le a függőségeket: `python setup_deps.py`
Program indítása: `python run_dev_win.py`

======== Használati Útmutató ========

A futtatás után a terminálban ki kell választani a futtatási opciókat. 
Új szimuláció futtatásához nyomj ENTER-t a run_dev terminálba, vagy zárd be, hogy leállítsa a folyamatokat,
melőtt újra elindítot a run dev fájlt
