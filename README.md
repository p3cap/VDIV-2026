## Setup

### Python env.:
```bash
cd backend
python -m venv venv
venv/bin/activate # 'source venv/bin/activate' for linux/mac
pip install -r requirements.txt
```

### Frontend (vue):
- Install [node.js](https://nodejs.org/en/download)
```bash
cd frontend
npm install
```
